# Руководство по созданию ИИ-бота для дизайна интерьера (аналог Ais Mebel)

Данное руководство описывает процесс создания Telegram-бота, который использует искусственный интеллект для визуализации мебели и дизайна интерьера на основе фотографий пользователей или эскизов.

---

## 1. Обзор функционала бота
Бот выступает в роли виртуального ИИ-дизайнера. Основные возможности:
*   **Визуализация по фото:** Пользователь загружает фото пустой комнаты, а ИИ «обставляет» её мебелью.
*   **Генерация по ТЗ:** Учет текстовых пожеланий (цвет, материалы, стиль, наличие подсветки и т.д.).
*   **Работа с эскизами:** Превращение нарисованных от руки чертежей в фотореалистичные рендеры.
*   **Точная геометрия:** Мебель вписывается в реальные габариты помещения благодаря технологиям контроля формы.

---

## 2. Технологический стек
Для реализации проекта рекомендуется использовать следующие инструменты:

| Компонент | Рекомендуемое решение | Примечание |
| :--- | :--- | :--- |
| **Интерфейс** | Telegram Bot API | Библиотека `aiogram` (Python) |
| **Язык разработки** | Python 3.10+ | Идеален для работы с ИИ и API |
| **ИИ-движок** | Google Gemini 1.5 Pro/Flash | Мультимодальная модель для генерации изображений по тексту и фото |

| **База данных** | PostgreSQL | Хранение истории запросов и подписок |

## 3. Пошаговый план разработки

### Шаг 1: Регистрация и настройка Telegram-бота
1.  Напишите `@BotFather` в Telegram и создайте нового бота.
2.  Получите **API Token**.
3.  Настройте меню команд: `/start`, `/generate`, `/sketch`, `/help`.

### Шаг 2: Настройка ИИ-генерации с Google Gemini (Ядро системы)
Благодаря мультимодальности Gemini, процесс контроля формы и генерации значительно упрощается.
1.  **Использование Gemini 1.5 Pro/Flash:**
    *   Gemini может принимать на вход как текстовые запросы, так и изображения (фото комнаты, эскизы).
    *   Модель способна понимать контекст изображения и генерировать мебель, учитывая геометрию и перспективу помещения без явного использования ControlNet.
2.  **Prompt Engineering (Промпт-инжиниринг):**
    *   Создайте «системный промпт», который будет добавляться к запросу пользователя, направляя Gemini на нужный стиль и качество.
    *   *Пример:* `(high quality, photorealistic, 8k, interior design, architectural photography, soft lighting, highly detailed textures, generate furniture that fits the room's perspective)`.

### Шаг 3: Написание Backend-части (Python)
Ваш код должен выполнять следующий цикл:
1.  **Прием данных:** Бот получает фото и текст от пользователя.
2.  **Обработка:** Если текст на русском, используйте библиотеку `googletrans` или API переводчика для перевода на английский (ИИ лучше понимает английский).
3.  **Запрос к ИИ:** Отправка фото комнаты (если есть) и промпта (ТЗ) к Gemini API.
    *   Gemini будет использовать изображение как контекст для генерации мебели, вписывая её в существующее пространство.
4.  **Выдача результата:** Получение ссылки на готовое изображение и отправка его пользователю в Telegram.

### Шаг 4: Обработка эскизов и чертежей
Для функции «оживления» чертежей:
1.  Пользователь присылает фото наброска.
2.  Бот отправляет фото наброска и текстовый запрос (например, "преврати этот эскиз в фотореалистичный интерьер с мебелью из дерева и металла") в Gemini API.
3.  Gemini интерпретирует эскиз и генерирует фотореалистичное изображение, накладывая текстуры и создавая объем.

### Шаг 5: Монетизация и деплой
1.  **Подписки:** Реализуйте систему лимитов (например, 3 бесплатные генерации, далее — платная подписка).
2.  **Оплата:** Интегрируйте Telegram Payments (через ЮKassa, Stripe или другие процессоры).
3.  **Сервер:** Разверните бота на VPS (например, Ubuntu Server). Если используете внешние API для ИИ, достаточно 2 ГБ ОЗУ.

---

## 4. Пример структуры промпта для бота
Когда пользователь пишет: *"Белая кухня с подсветкой в стиле минимализм"*, бот должен отправить в ИИ-движок:
> **Prompt:** `Minimalist white kitchen, LED under-cabinet lighting, marble countertop, handleless cabinets, built-in appliances, photorealistic, 8k, interior design magazine style, sharp focus, cinematic lighting.`
> **Negative Prompt:** `blurry, distorted, low quality, messy, unrealistic colors, floating furniture.`
